# Clear-DnsClientCache

## SYNOPSIS
Clears the contents of the DNS client cache.

## SYNTAX

```
Clear-DnsClientCache [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
The **Clear-DnsClientCache** cmdlet deletes all the contents of the DNS client cache.
Running this cmdlet is equivalent to running `ipconfig /flushdns`.

## EXAMPLES

### Example 1: Clear the DNS client cache
```
PS C:\> Clear-DnsClientCache
```

This command clears the DNS client cache.
