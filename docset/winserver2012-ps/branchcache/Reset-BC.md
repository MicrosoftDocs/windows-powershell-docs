# Reset-BC

## SYNOPSIS
Resets BranchCache to the default configuration.

## SYNTAX

```
Reset-BC [-ResetFWRulesOnly] [-ResetPerfCountersOnly] [-Force] [-CimSession <CimSession[]>]
 [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Reset-BC** cmdlet resets BranchCache to a default configuration.

## EXAMPLES

### EXAMPLE 1
```
PS C:\>Reset-BC
```

This example resets BranchCache to the default configuration.
