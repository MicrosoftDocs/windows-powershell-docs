# Get-WebsiteState

## SYNOPSIS
Gets the state of an IIS Web site.

## SYNTAX

```
Get-WebsiteState [[-Name] <String>] [<CommonParameters>]
```

## DESCRIPTION
Gets the state of an IIS Web site.

## EXAMPLES

### -------------- EXAMPLE 1: Get the state of a Web site --------------
```
IIS:\>Get-WebsiteState -Name "Default Web Site"
```

Returns the state of the Default Web Site.

Value

-----

Started

## PARAMETERS
