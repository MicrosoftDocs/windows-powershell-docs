# Clear-BCCache

## SYNOPSIS
Deletes all data in all data cache files and all hash cache files.

## SYNTAX

```
Clear-BCCache [-Force] [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
The **Clear-BCCache** cmdlet deletes all data in all cache files.

## EXAMPLES

### EXAMPLE 1
```
PS C:\>Clear-BCCache
```

This example deletes all content and content information from all cache files.
