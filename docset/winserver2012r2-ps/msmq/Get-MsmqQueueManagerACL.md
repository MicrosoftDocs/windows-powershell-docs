# Get-MsmqQueueManagerACL

## SYNOPSIS
Returns access control lists for the local queue manager.

## SYNTAX

```
Get-MsmqQueueManagerACL [<CommonParameters>]
```

## DESCRIPTION
The **Get-MsmqQueueManagerACL** cmdlet returns an array of **MsmqQueueManagerAcl** objects for the local queue manager.

## EXAMPLES

### Example 1
```
PS C:\>Get-MsmqQueueManagerACL
```
