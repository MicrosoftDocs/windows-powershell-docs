# Remove-NetNatStaticMapping

## SYNOPSIS
Removes static mappings from a NAT instance.

## SYNTAX

### Query (cdxml) (Default)
```
Remove-NetNatStaticMapping [-StaticMappingID <UInt32[]>] [[-NatName] <String[]>] [-CimSession <CimSession[]>]
 [-ThrottleLimit <Int32>] [-AsJob] [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject (cdxml)
```
Remove-NetNatStaticMapping -InputObject <CimInstance[]> [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>]
 [-AsJob] [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Remove-NetNatStaticMapping** cmdlet removes one or more static mappings configured on network address translation (NAT) instances.
A static mapping enables an incoming connection from an external network to an internal network through the NAT.

## EXAMPLES

### Example 1: Remove all static mappings from a NAT instance
```
PS C:\> Remove-NetNatStaticMapping
```

This command removes all static mapping of NAT instances configured on the local computer.
