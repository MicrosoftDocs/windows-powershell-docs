# Remove-SMServerPerformanceLog

## SYNOPSIS
Removes Server Manager performance log files.

## SYNTAX

```
Remove-SMServerPerformanceLog -CollectorName <String> [-ThresholdMSec <UInt64>] [-CimSession <CimSession[]>]
 [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Remove-SMServerPerformanceLog** cmdlet removes the Server Manager performance log files.

## EXAMPLES

### Example 1:
```
PS C:\>Remove-SMServerPerformanceLog -CollectorName $ContosoPerf
```

This command removes the performance logs for the data collector named by the variable **$ContosoPerf**.
