# Stop-VMTrace

## SYNOPSIS
Stops tracing to file.

## SYNTAX

```
Stop-VMTrace [<CommonParameters>]
```

## DESCRIPTION
The **Stop-VMTrace** cmdlet stops tracing to a file.

## EXAMPLES

### Example 1: Stop tracing
```
PS C:\> Stop-VMTrace
```

This command stops tracing.
