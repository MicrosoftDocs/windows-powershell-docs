# Export-WinhttpProxy

## SYNOPSIS
Exports the current Winhttp proxy configuration.

## SYNTAX

```
Export-WinhttpProxy
```

## DESCRIPTION

Creates a script that contains the current configuration. If saved to a file, this script can be
used to restore altered configuration settings.
