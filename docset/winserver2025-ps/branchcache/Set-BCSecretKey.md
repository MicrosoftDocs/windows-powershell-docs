# Set-BCSecretKey

## SYNOPSIS
Sets the cryptographic key used in the generation of segment secrets.

## SYNTAX

```
Set-BCSecretKey [[-Passphrase] <String>] [-Force] [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>]
 [-AsJob] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Set-BCSecretKey** cmdlet sets the cryptographic key that is used in the generation of segment secrets.
Use this cmdlet when you deploy BranchCache-enabled content servers in a cluster or behind a network load balancer.
If a file or webpage exists on multiple content servers, each server must use the same secret key.
Otherwise, each copy of the file will be cached separately in the branch office.

## EXAMPLES

### Example 1: Set a cryptographic key
```
PS C:\> Set-BCSecretKey -Passphrase "mySecretPhrase"
```

This command sets the cryptographic key that is used in the generation of segment secrets, using the pass phrase mySecretPhrase.
