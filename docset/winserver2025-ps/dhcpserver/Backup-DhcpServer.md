# Backup-DhcpServer

## SYNOPSIS
Backs up the DHCP database of DHCP server service.

## SYNTAX

```
Backup-DhcpServer [-ComputerName <String>] [-Path] <String> [-CimSession <CimSession[]>]
 [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Backup-DhcpServer** cmdlet backs up the Dynamic Host Configuration Protocol (DHCP) database of DHCP server service to a specified location.

## EXAMPLES

### Example 1: Back up the DHCP server database
```
PS C:\> Backup-DhcpServer -ComputerName "dhcpserver.contoso.com" -Path "C:\Windows\system32\dhcp\backup"
```

This example backs up the DHCP server database to the specified location.
