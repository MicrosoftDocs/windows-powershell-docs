# Remove-DhcpServerv4Failover

## SYNOPSIS
Removes failover relationships.

## SYNTAX

```
Remove-DhcpServerv4Failover [-Name] <String[]> [-Force] [-PassThru] [-ComputerName <String>]
 [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Remove-DhcpServerv4Failover** cmdlet removes one or more specified failover relationships.
This cmdlet removes the failover relationship from both of the computers that run Dynamic Host Configuration Protocol (DHCP) server services which are part of this failover relationship.

## EXAMPLES

### Example 1: Delete a failover relationship
```
PS C:\> Remove-DhcpServerv4Failover -ComputerName "dhcpserver.contoso.com" -Name "SFO-SIN-Failover"
```

This example deletes the failover relationship named SFO-SIN-Failover from the DHCP server service that runs on the computer named dhcpserver.contoso.com.
