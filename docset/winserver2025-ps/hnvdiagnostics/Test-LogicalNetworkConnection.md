# Test-LogicalNetworkConnection

## SYNOPSIS
Tests a logical network connection.

## SYNTAX

```
Test-LogicalNetworkConnection [<CommonParameters>]
```

## DESCRIPTION
The **Test-LogicalNetworkConnection** cmdlet tests whether a logical network can be reached.
