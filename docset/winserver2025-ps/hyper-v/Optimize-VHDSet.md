# Optimize-VHDSet

## SYNOPSIS
Optimizes VHD set files.

## SYNTAX

```
Optimize-VHDSet [-Path] <String[]> [-AsJob] [-Passthru] [-CimSession <CimSession[]>] [-ComputerName <String[]>]
 [-Credential <PSCredential[]>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Optimize-VHDSet** cmdlet optimizes the allocation of space used by virtual hard disk (VHD) set files.

## EXAMPLES

### Example 1: Optimize a VHD set
```
PS C:\> Optimize-VHDSet -Path "c:\data.vhds"
```

This command optimizes the allocation of space on the VHD set file c:\data.vhds.
