# Save-VM

## SYNOPSIS
Saves a virtual machine.

## SYNTAX

### Name (Default)
```
Save-VM [-CimSession <CimSession[]>] [-ComputerName <String[]>] [-Credential <PSCredential[]>]
 [-Name] <String[]> [-AsJob] [-Passthru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### VMObject
```
Save-VM [-VM] <VirtualMachine[]> [-AsJob] [-Passthru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Save-VM** cmdlet saves a virtual machine.
This is similar to hibernating a physical computer.

## EXAMPLES

### Example 1
```
PS C:\> Save-VM -Name TestVM
```

Saves virtual machine TestVM.

### Example 2
```
PS C:\> Get-VM -Name Win* | Save-VM
```

Saves all virtual machines whose name starts with Win.
