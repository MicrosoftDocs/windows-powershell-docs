# Set-StorageQosPolicyStore

## SYNOPSIS
Sets the fields on the policy store object.

## SYNTAX

```
Set-StorageQosPolicyStore [-InputObject <CimInstance[]>] [[-IOPSNormalizationSize] <UInt32>]
 [-CimSession <CimSession[]>] [-ThrottleLimit <Int32>] [-AsJob] [-PassThru] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIPTION
The **Set-StorageQosPolicyStore** cmdlet sets the fields on the policy store object.

## EXAMPLES

### Example 1: Set the normalized IO to a specified size
```
PS C:\>Set-StorageQosPolicyStore -IOPSNormalizationSize 2048
```

This command sets the normalized IO size to 2KB.
